Meus Projetos

MeuTicket

MeuTicket é um sistema de gestão de demandas desenvolvido com Django, projetado para otimizar o fluxo de trabalho e a comunicação entre operadores e a equipe de suporte. O sistema visa melhorar a organização, eficiência e comunicação nas operações de suporte, demonstrando habilidades em desenvolvimento web com Django.

JC Burguer

Aplicação Web para Compra de Lanches Online. Desenvolvido com HTML, Tailwind CSS e JavaScript, este projeto facilita a experiência de compra de lanches online. A landing page integra a API do WhatsApp para o envio direto ao proprietário das informações do pedido, incluindo itens, entrega e pagamento.

BragaMusic

BragaMusic: Aplicação Web Full-Stack para Venda de Instrumentos Musicais. Construído com Python no backend (Django) e HTML, CSS e JavaScript no frontend, o BragaMusic simula uma plataforma de e-commerce para instrumentos musicais. Uma funcionalidade chave é a integração com a API do Mercado Pago, demonstrando a capacidade de implementar fluxos de pagamento online de forma segura e integrada.
